# Программа опросник
Эта программа предлагает пройти краткий опрос по языку Java

Установка:
1. Клонируйте этот репозиторий через git clone
2. Скомпилируйте Java файлы (javac Mein1.java TriviaMaster.java)
3. Запустите программу (java Mein1)
